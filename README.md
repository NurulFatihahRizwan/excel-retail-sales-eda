# Exploratory Data Analysis of Retail Sales Data (Excel)

## Overview
This project presents an exploratory data analysis (EDA) of retail sales data using Microsoft Excel. The analysis focuses on understanding customer demographics, purchasing behavior, and revenue trends to derive actionable business insights.

## Dataset
The dataset contains approximately 200 retail transaction records, including:
- Customer age and gender
- Product categories
- Purchase amount (RM)
- Purchase dates

## Analysis Performed
- Data cleaning (handling missing values, removing duplicates)
- Descriptive statistics
- Customer segmentation by age group and gender
- Revenue and product category analysis
- Purchase amount band analysis
- Monthly sales trend analysis
- Pivot tables and charts were used extensively for analysis and visualization

## Key Insights
- Customers aged 25–34 form the largest customer segment
- Customers aged 35–44 have the highest average purchase amount
- Medium-value purchases account for 56% of transactions
- Electronics is the highest revenue-generating product category
- Sales show seasonal fluctuations across months

## Tools Used
- Microsoft Excel
